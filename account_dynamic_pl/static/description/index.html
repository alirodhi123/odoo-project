Dynamic Profit and Loss Report with drill down - Community Edition

Do you want to show P and L report in screen and inspect it? don't worry here we provide a plugin with easy option to show P and L Report on the screen to analyze without download into system. You have an interactive filter pannel as well as result screen with much more options. Also can download report into Pdf as well as Xlsx report.


Interactive filter pannel

Report welcomes you with a very beautiful filter section


After selecting appropriate filter, press 'Apply' button to get details of P and L.

Here you can get details of each row by clicking.


Based on filters selected, It will show different results


Download report in Pdf and Xlsx formats.


Report view structure is exactly same as that it configured on the backend. ie Financial reports > Account Reports. Any changes on this menu will reflect on this report structure as well as pdf and xlsx reports. Profit entry in this report is not there in the default Odoo DB.
