Elsaka Training



Overview,



This module will be helpful for training centers ,academies and companies to manage training and courses.





- Define course details:



1- Course name (Python course).

2- Course Price (50 $).

3- Course code (PY).

4- Course content (course description).

Access Right

The normal employee will see his course requests only.

The HR manager will see all employees course requests

Validation:-

Only available Courses will be shown when user create Course request.

- Trainers:



Trainers are retrieved from partners.

In partner screen there is check box (Is a trainer) make it checked to define the partner as a trainer.



-Course schedule



-Choose course

-Define the course duration.

-Choose tags to enable employees who are related to the same tag to see the course schedule.

-Choose the trainer (will be retrieved from trainer from defined trainers).

-Define course capacity (4 persons).

-Define the course agreement.

-There is a work flow for the course (new or active or closed), make the course in (Active) state to notify employees who related to that course.



-Course Request



-Employee request a specific course then the system will retrieve course price,content and agreement automatically.

-Employee send the request to his manager to be approved.

-In employee profile there is a smart button (Courses) to view all employee course requests.
